ConnorsAerials: Personal Photography Website

Project Information

  • Built a multipage personal photography website to increase exposure and highlight successes of Student Athletes at Johns Hopkins University.
  • Built a responsive multi-page photography portfolio website leveraging HTML and advanced CSS techniques, including Flexbox, Grid, and media queries, ensuring seamless functionality and an elegant design across all device types.
  • Developed a Python based image search engine utilizing TF-IDF and Cosine Similarity to allow users to search for images based off natural language queries to enhance accuracy.
