Visilant: Telemedicine Startup

Project Information

  • Hopkins Medical startup company leveraging AI, machine learning, and telemedicine app to create a system designed aid in the diagnosis of eye conditions in rural India.
  • Designed and developed the telemedicine app using Figma and React Native. The app has helped improve healthcare accessibility and access to high quality eye treatment for over 1,000 individuals in rural India.
  • Increased user efficiency from previous designs by over 50%. This was achieved through an intuitive navigational flow based off user research and interviews. I managed screen transitions and routing logic with stack navigators, state variables, and effect hooks.
  • Drove app scalability by designing and developing a modular, dynamic questionnaire system using JSON for patient symptom input, built intuitive UI components, and implemented patient search functionality to filter by relevant criteria.
